# MAS (Model Architecture Settings)

This repository contains configuration files for OCR model training and evaluation, specifically focused on Arabic and Uzbek scripts for the ICDAR 2026 competition.

## Repository Structure

- `configs/`: Directory containing model configuration files (YAML) for various OCR frameworks including EasyOCR, PaddleOCR, and MMOCR.

## Purpose

The aim of this project is to fine-tune and evaluate SOTA OCR models on historical and multi-script datasets.
